# 8.python

Python packaging and documentation layer for the UberClock LiteX SoC.

Key entry points:

- `src/uberclock_soc/` contains the reusable package modules.
- `python -m uberclock_soc ...` runs the SoC build entry point.
- `pip install -e 2.soc/8.python` installs the package in editable mode.
- `0.doc/docs/` is the primary Sphinx site used for GitHub Pages publication.

uberclock_soc
=============

Package overview
----------------

``uberclock_soc`` is the import root for the AX7203 LiteX integration package.
Most useful public functionality is reached through the submodules documented
below, especially ``soc.py`` for build entry and ``ubddr3.py`` /
``uberclock_core.py`` for major integration blocks.
